PyConju
========

Guide
===============

.. _PyConju:

**PyConju** is a Python package for merging multiple files into a single file. 

It is designed to simplify your data science workflow! 🎉🐍

.. note::

   💻 **Try the new** `PyConju Web App`_!

.. _PyConju Web App: https://pyconju.pythonanywhere.com

The Ultimate Data Science File Merger 📊🔀

With pyconju, you can effortlessly merge multiple file formats (XLSX, XLS, CSV) into a single file,
saving you valuable time and effort. It's the go-to tool for data analysts, scientists, and anyone working with complex datasets.


View difPy on `GitHub <https://github.com/dyagee/pyconju>`_ and `PyPi <https://pypi.org/project/pyconju/>`_.
