PyConju Features

✅ Seamlessly merge multiple files into a single file with just a few lines of code.
✅ Support for popular file formats like XLSX, XLS, and CSV.
✅ Flexibility to handle large datasets efficiently.
✅ Customizable options for merging strategies and column mappings.

Whether you’re dealing with large-scale data integration projects or need to combine multiple reports, pyconju has got you covered!

Give it a try today by installing via pip:

(.venv) $ pip install pyonju

Tracking Bugs

If you run into issues, do well to visit the Contributing to pyconju section for more details.

If you have already read the guidlines, log the issue directly on the github repo.
